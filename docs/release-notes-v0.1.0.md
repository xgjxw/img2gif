# img2gif v0.1.0

首个 Python 版本发布。

## 这次更新

- 新增跨平台桌面版图片转 GIF 工具
- 支持拖拽图片文件到窗口
- 支持剪贴板图片导入
- 支持剪贴板文本路径 / 图片 URL 导入
- 支持剪贴板结果复制
- 支持清空当前状态、打开当前结果、查看最近结果
- 支持模式 1 / 模式 2 两种 GIF 量化方式
- 支持 GIF / PNG 两种输出格式
- 支持原始 / 40 / 80 / 120 / 200 五种输出尺寸
- 转换结果自动保存到桌面 `pic2meme-output`
- 转换成功后自动打开文件管理器并定位输出文件
- UI 已按桌面工具场景做过重新整理
- 已支持推送 tag 后自动创建 GitHub Release

## 平台支持

- Windows：提供自动构建的 `img2gif.exe`
- macOS：提供 GitHub Actions 自动构建产物
- Linux：支持源码运行

## 发布说明

- 本版本不依赖旧版“复制文件到剪贴板”的单一路径
- 输入和输出都补充了更完整的剪贴板支持
- 仓库不提交 `.exe` / `.zip`，构建产物由 GitHub Actions 自动生成

## 使用方式

1. 下载发布页里的 Windows 或 macOS 压缩包
2. 解压后运行 `img2gif`
3. 拖拽图片到窗口，或使用系统粘贴快捷键导入图片

## 已知说明

- 首次运行如果被系统拦截，请手动允许
- 不同聊天软件对 GIF 剪贴板支持程度不同
- 输出目录固定为桌面 `pic2meme-output`
