# img2gif v0.1.0

首个 Python 版本发布。

## 这次更新

- 新增 Windows 桌面版图片转 GIF 工具
- 支持拖拽图片文件到窗口
- 支持 `Ctrl+V` 粘贴剪贴板图片
- 支持从剪贴板 HTML 中提取图片地址
- 支持模式 1 / 模式 2 两种 GIF 量化方式
- 支持原始 / 40 / 80 / 120 / 200 五种输出尺寸
- 转换结果自动保存到桌面 `pic2meme-output`
- 转换成功后自动打开资源管理器并定位输出文件

## 发布说明

- 本版本不依赖“复制文件到剪贴板”，避开了原始方案里常见的剪贴板失败问题
- 仓库不提交 `.exe`，可执行文件由 GitHub Actions 自动构建

## 使用方式

1. 下载发布页里的 Windows 压缩包
2. 解压后双击 `pic2meme-py.exe`
3. 拖拽图片到窗口，或按 `Ctrl+V` 粘贴图片

## 已知说明

- 首次运行如果被 Windows 拦截，请手动选择继续运行
- 输出目录固定为桌面 `pic2meme-output`
